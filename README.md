#### Communication Protocols

* [WiFi](https://www.wi-fi.org/) -  Allowing nearby digital devices to exchange data by radio waves
* [Bluetooth](https://www.bluetooth.com/) - Short-range wireless data exchange technology standard
* [Zigbee](https://zigbeealliance.org/) - Low-cost, low-power, wireless mesh network standard
* [MQTT](https://mqtt.org/) - Lightweight message protocol for sending simple data flows from sensors to applications and middleware
* [OPC-UA](https://opcfoundation.org/) - OPC UA  is one of the most important communication protocols for Industry 4.0 and the IoT
* [LoRa](https://lora-alliance.org/) - Long Range proprietary low-power wide-area network modulation technique
* [WebSocket](https://en.wikipedia.org/wiki/WebSocket) - Providing full-duplex communication channels over a single TCP connection
* [Cellular](https://en.wikipedia.org/wiki/Cellular_network) - Capable of facilitating massive flows of data
  * [GSM](https://en.wikipedia.org/wiki/GSM) - The Global System for Mobile Communications
  * [GPRS](https://en.wikipedia.org/wiki/General_Packet_Radio_Service) - General Packet Radio Service
  * [EDGE 2G](https://en.wikipedia.org/wiki/2G) - Second-generation cellular network
  * [UMTS/HSPA 3G](https://en.wikipedia.org/wiki/3G) - Third generation of wireless mobile telecommunications technology
  * [LTA 4G](https://en.wikipedia.org/wiki/4G) - Fourth generation of broadband cellular network technology
